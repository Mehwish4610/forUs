# Landing Page UX

**Project:** forUs  
**Version:** 0.1.0  
**Status:** Draft  
**Last Updated:** 29 June 2026

---

# Purpose

The landing page introduces users to forUs and should answer three questions within the first few seconds:

1. What is forUs?
2. Why should I trust it?
3. How do I start?

The experience should feel calm, modern, welcoming, and effortless.

---

# User Goals

A visitor should be able to:

- Understand what forUs is.
- Create a room.
- Join an existing room.
- Continue as a guest.
- Sign in with Google.
- Learn how the platform works.

---

# Design Principles

The landing page follows the Aurora Design System.

Design priorities:

- Clean
- Spacious
- Fast
- Minimal
- Accessible
- Mobile-first
- Privacy-first

---

# Sections

The landing page contains nine sections.

1. Hero
2. Privacy Promises
3. How It Works
4. Interactive Preview
5. Why forUs
6. Personalization
7. Security
8. FAQ
9. Footer

These sections should appear in this order.

## Hero Animation

On page load:

1. Aurora fades in.
2. Navigation slides down.
3. Hero text fades upward.
4. Buttons appear.
5. Application preview fades in.

Total animation duration:

Approximately 1 second.

Animations should feel smooth and calm.

## Accessibility

Buttons must be keyboard accessible.

The heading should use a single H1 element.

All interactive elements require visible focus indicators.

Animations must respect reduced-motion preferences.

- How to begin.

# Section 8 — Frequently Asked Questions

## Goal

Answer the most common questions before users need to ask them.

---

## Questions

### Do I need an account?

No. You can continue as a guest or optionally sign in with Google.

---

### Are chats stored permanently?

No. Messages follow the room's chat retention policy.

---

### Can I change my username?

Yes. Your display name can be changed inside each room.

---

### Can I use forUs on my phone?

Yes. forUs is fully responsive and can be installed as a Progressive Web App (PWA).

---

### Can I create multiple rooms?

Yes. You can create and manage multiple rooms from your dashboard.

---

## Layout

Accordion style.

Only one answer is expanded at a time.
