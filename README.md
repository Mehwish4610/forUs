# forUs

> A privacy-first communication platform built for meaningful conversations.

## Vision

forUs is designed to provide secure, temporary and private communication through rooms, realtime messaging and voice/video calling while maintaining a calm, distraction-free experience.

## Tech Stack

### Frontend
- React 19
- TypeScript
- Vite
- Tailwind CSS v4

### Backend
- Express
- TypeScript
- Zod

### Tooling
- pnpm Workspace
- Turborepo
